How It Works

Start tracking in 3 steps

No sign-up required. Just install, enable notifications, and spend normally.

1

Install SpendWise

Download the APK from GitHub and install on your Android phone. Takes 30 seconds.

2

Enable Notifications

Grant notification access so SpendWise can read UPI payment alerts. We only read financial notifications.

3

Spend Normally

Every UPI payment automatically appears in your dashboard, categorized by AI. Check reports anytime.

Installation Note

Since SpendWise is installed directly (not from Play Store), Google Play Protect may show a warning. This is normal for all sideloaded apps that read notifications and SMS for expense tracking.

To install: Tap "More details" → "Install anyway". SpendWise is fully open-source — audit the code yourself.
